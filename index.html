Mission

Survive 5 nights

  • Players: 1–5 Ghost commandos
  • Cycle: 22 minute days / 22 minute nights
  • Pressure: wave-based Zerg attacks
  • Death: respawn, lose resources and structures, keep progression

Core loop

Reconnoiter. Commit. Extract.

Each phase lasts 22 minutes. The cycle below shows the field rhythm; the cards summarize the operating rules that matter most.

Hotbar focus

Three active commands, many passive systems

Heal, Speed, and Stun remain universal commands. Weapon packages carry the specialized target abilities.

Heal Self-save → ally burst → station/area pulse
Speed Escape sprint → squad reposition → reload/attack tempo window
Stun Critter catch → Maelstrom trap → elite interrupt/control field
